Here is a set of Python code that should help you with your project: ```python import pandas as pd import numpy as np from sklearn.preprocessing import StandardScaler from sklearn.model_selection import train_test_split from sklearn.linear_model import LinearRegression from sklearn.neural_network import MLPRegressor from sklearn.svm import SVR from sklearn.metrics import mean_squared_error, r2_score import matplotlib.pyplot as plt import seaborn as sns # i) Download the dataset and consider 'pH' as the dependent variable df = pd.read_csv('winequality-red.csv') X = df.drop('pH', axis=1) y = df['pH'] # ii) Draw plots to find data normality, outliers, biasedness using 'ggplot' sns.set(style='whitegrid', palette='muted') sns.barplot(x='quality', y='pH', data=df) plt.show() sns.distplot(df['pH']) plt.show() df['pH'].value_counts().plot(kind='pie', autopct='%1.1f%%') plt.show() sns.boxplot(x='quality', y='pH', data=df) plt.show() # iii) Use over sampling method to handle imbalanced dataset from imblearn.over_sampling import SMOTE sm = SMOTE(random_state=42) X_res, y_res = sm.fit_resample(X, y) # iv) Apply label and one hot encoding to different variables X_res = pd.get_dummies(X_res) # v) Apply standard scalar operation on the dataset scaler = StandardScaler() X_res = scaler.fit_transform(X_res) # vi) Perform application of linear regression, ANN and SVR algorithms X_train, X_test, y_train, y_test = train_test_split(X_res, y_res, test_size=0.2, random_state=42) lr = LinearRegression() lr.fit(X_train, y_train) y_pred_lr = lr.predict(X_test) ann = MLPRegressor(hidden_layer_sizes=(64,64), max_iter=1000) ann.fit(X_train, y_train) y_pred_ann = ann.predict(X_test) svr = SVR() svr.fit(X_train, y_train) y_pred_svr = svr.predict(X_test) # vii) Evaluate and compare performances of models using RMSE, R-squared and adjusted R-squared methods in a single table rmse_lr = np.sqrt(mean_squared_error(y_test, y_pred_lr)) r2_lr = r2_score(y_test, y_pred_lr) adj_r2_lr = 1 - (1-r2_lr)*(len(y)-1)/(len(y)-X.shape[1]-1) rmse_ann = np.sqrt(mean_squared_error(y_test, y_pred_ann)) r2_ann = r2_score(y_test, y_pred_ann) adj_r2_ann = 1 - (1-r2_ann)*(len(y)-1)/(len(y)-X.shape[1]-1) rmse_svr = np.sqrt(mean_squared_error(y_test, y_pred_svr)) r2_svr = r2_score(y_test, y_pred_svr) adj_r2_svr = 1 - (1-r2_svr)*(len(y)-1)/(len(y)-X.shape[1]-1) results_df = pd.DataFrame({'Model': ['Linear Regression', 'ANN', 'SVR'], 'RMSE': [rmse_lr, rmse_ann, rmse_svr], 'R-squared': [r2_lr, r2_ann, r2_svr], 'Adjusted R-squared': [adj_r2_lr, adj_r2_ann, adj_r2_svr]}) print(results_df) ``` This code should help you download the wine quality dataset and consider the pH value as the dependent variable. It also includes code to draw barplots, histograms, pie charts and boxplots using the `ggplot` package to find data normality, outliers and biasedness. The code also includes steps to handle imbalanced datasets using oversampling methods and apply label and one-hot encoding to different variables. It also applies standard scalar operation on the dataset and performs linear regression, ANN and SVR algorithms. Finally it evaluates and compares the performances of the models using RMSE, R-squared and adjusted R-squared methods in a single table. I hope this helps!
